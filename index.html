user experience user interface

내가 생각하는 사용자경험

UX에 대해서 다양한 리서치를 해보면서 사용자의 경험을
바탕으로 어떤 제품이나 서비스를 구축하는 것의 중요성을 깨달았다. 내가 생각하는 UX란,
사용자가 원하는 결과물을 얻기위한 과정중에서 불편함을 최소화하면서,
선호하고, 쉽고 즐겁게 사용할 수 있는 서비스를 제공하는 것이라 생각한다.

메세지를 담아내는 디자인

UI만 화려하다고해서 좋은 디자인은 아닙니다.
사용자의 경험이 바탕에 없다면 그 서비스는 지속되지 못할 것입니다. UI Design이란,
사용자와 시각적으로 드러나는것들의 소통이라고 생각합니다.
레이아웃, 디자인, 색상 등 마주보게 되는것을 어떤 방식으로
이용하도록 만드는 것이 UI Design이라 생각합니다.

UX와 UI를 조화롭게 디자인하여
사용자에게 좋은 경험을 선사할 수 있는 인재로 발전해 나가겠습니다.
